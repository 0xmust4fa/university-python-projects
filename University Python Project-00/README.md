University Python Project-00

This project includes three beginner-level Python programs:

1. Welcome Program for Students
Takes the student's name, last name, and student ID as input and prints a welcome message.

2. Average Counting Program
Takes 3 subject scores as input and calculates the average.

3. Rectangle Environment and Area
Takes length and width of a rectangle and calculates:
- محیط (Environment / Perimeter)
- مساحت (Area)


Files:
- welcome.py
- average.py
- rectangle.py
