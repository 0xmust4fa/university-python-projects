University Python Project 05

This project contains a collection of simple yet practical Python functions and programs designed for beginners.

Features

1. Student GPA Calculator
- Takes a student's name and three grades.
- Calculates and displays the average.
- *File:* `student_gpa.py`

2. Count 'A' or 'a' in Text
- Accepts a text string.
- Returns the number of times 'A' or 'a' appears.
- *File:* `count_a.py`

3. Generate Even & Odd Numbers
- Takes a number `N`.
- Returns two lists:
  - Even numbers up to `N`
  - Odd numbers up to `N`
- *File:* `even_odd_lists.py`

4. Simple Calculator (Menu-Based)
- Offers a menu with:
  - Addition 
  - Subtraction 
  - Multiplication 
  - Division 
  - Exit 
- Loops until the user selects "Exit".
- Each operation is handled by a separate function.
- *File:* `menu_calculator.py`

How to Run

python <filename>.py
