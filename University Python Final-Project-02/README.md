Item Management System (Login Protected)

This is a *command-line Python application* for managing a list of items with a simple *login system*. It demonstrates user authentication, item handling (add, remove, display, clear), and modular programming using Python files.

---

Project Structure

- *main.py* – Main menu and program controller  
- *login.py* – Handles user login (with max 3 attempts)  
- *items.py* – Functions to manage the items list

---

Login Credentials

- *Username*: `admin`

---

Features

- Login system with limited attempts
- Add new items
- Remove existing items
- Show current items
- Clear all items
- Exit option

---

How to Run

1. Make sure all three `.py` files are in the same directory:
   - `main.py`
   - `login.py`
   - `items.py`

2. Run the program:

```bash
python main.py
```

3. Enter the login credentials when prompted.
